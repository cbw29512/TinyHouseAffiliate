# Definition of Done

## Project

TinyHouseAffiliate

## First Production Milestone

The first production milestone is a safe, GitHub-native tiny-home affiliate automation scaffold that can collect approved provider data, validate state, and prepare content assets without publishing anything publicly or spending money.

## Done Means

- The repository has a clear README.
- The data schema is defined before automation logic expands.
- Provider data can be stored in JSON.
- Example provider data exists.
- A validation script can check the expected state structure.
- No secrets are committed.
- No public publishing happens without human approval.
- Affiliate terms, provider image permissions, and disclosure requirements are documented before scraper automation is expanded.

## Not Done Yet

- No autonomous publishing.
- No paid API usage.
- No provider scraping beyond approved/read-only research.
- No use of provider images unless permission or terms are verified.
- No affiliate-link injection until approved links are confirmed.

## Human Approval Required Before

- Publishing videos.
- Uploading to YouTube, TikTok, Reels, or Shorts.
- Using provider images.
- Adding affiliate links.
- Running scheduled scraping.
- Spending money.
